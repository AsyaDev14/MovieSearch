# MovieSearch

MovieSearch is a web application that allows users to search for information
about movies and read reviews about them.

## Key Features

- **Movie Search**: Users can enter the title of a movie into the search field
  and receive a list of movies matching their query.
- **View Movie Information**: After selecting a movie from the search results,
  users can view detailed information about it, such as rating, description,
  genres, and cast.
- **Read Reviews**: Users can browse reviews about the movie left by other users
  to get an idea of its quality and content.

## Technologies

- **React**: The application is developed using the React library to create a
  dynamic user interface.
- **React Router**: React Router is used for navigation between pages of the
  application.
- **TMDB API**: The Movie Database (TMDB) API is used to fetch information about
  movies and reviews.

## Installation

1. Clone the repository using the `git clone` command.
2. Install dependencies by running `npm install`.
3. Start the application by entering `npm start`.
